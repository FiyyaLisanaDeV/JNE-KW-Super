# Pelacakan

Public tracking tersedia tanpa login.

Route:

```text
/tracking
/tracking/{receipt_number}
/t/{public_tracking_token}
```

Halaman publik menampilkan:

- Nomor resi
- Status terakhir
- Rute
- Tanggal check-in
- Estimasi tiba
- Timeline status
- Nama dan nomor HP yang sudah dimasking

Halaman publik tidak menampilkan:

- Alamat lengkap
- Nomor HP lengkap
- Catatan internal
- Detail biaya internal
